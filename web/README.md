# YANDLE Web

Web app for YANDLE — AI voice & chat for your business (landing, onboarding, dashboard, shareable link).

## Setup

```sh
npm i
npm run dev
```

Runs at [http://localhost:8080](http://localhost:8080).

## Build & deploy

```sh
npm run build
npm run preview   # local preview of production build
```

Deploy the `dist/` output to your host (e.g. Vercel, or `npm run deploy` for Vercel CLI).

## Stack

- Vite, React, TypeScript
- shadcn-ui, Tailwind CSS
- React Router, Framer Motion
