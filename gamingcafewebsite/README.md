# Gaming Cafe Website

Gaming cafe template site (Vite + React + TypeScript + shadcn-ui).

## Setup

```sh
npm i
npm run dev
```

Runs at [http://localhost:8080](http://localhost:8080).

## Build

```sh
npm run build
npm run preview
```

## Tests

```sh
npm run test        # Vitest
npx playwright test  # Playwright (start dev server first)
```
